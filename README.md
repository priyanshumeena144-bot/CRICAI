### 🚀 Real-time AI tactical engine for Bowlers, Batsmen & Coaches

CricketMind AI is an advanced AI-powered cricket analytics and decision-making platform designed for IPL teams, coaches, analysts, bowlers, and batsmen.

The system analyzes historical and live cricket data to generate:
- Bowling strategies
- Batting strategies
- Matchup intelligence
- Real-time tactical recommendations
- Win probability predictions
- Field placement suggestions

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/CricketMind-AI.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
